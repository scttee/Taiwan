Put new photos from your phone into this folder first.

Suggested road workflow:
1. AirDrop or save photos from your iPhone into this `uploads/` folder.
2. Run `npm run sort-photos`.
3. The script will group photos into `assets/photos/day-001/`, `day-002/`, etc.
4. It also writes `content/photo-manifest.json` so you can see what landed where.

After that, I can help wire the selected images into each day page and captions.
